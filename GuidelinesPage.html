IMPORTANT: The script will only run if AutoHotkey v2.xx and v1.xx are installed.

  • You can find the AutoHotkey v2.xx here: Download AutoHotkey v2.xx
  • STEP 1: Install this only for the current user.
  • STEP 2: Open double-click the script Lazy Agent 1.01.xx. It should prompt you to download AutoHotkey v1, it will install by itself.
  • STEP 3: Once you run the script, it will ask you for your master == support for the day.
  • STEP 4: Enter your First name and Initial.
  • You can test the name and support if they are correctly set by typing the following: "hdtmp" and "dictmp"

    • NOTE: While it works for a regular Notepad, we suggest to type the script on Notepad++
